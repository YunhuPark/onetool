인스타 맞팔 판독기

누가 내 팔로우를 끊었을까?
ZIP 파일 그대로 올려서 확인하세요.

📂 데이터 다운로드 방법 (필독)
  1. 인스타 설정 → 계정 센터 → 내 정보 및 권한
  2. 내 정보 내보내기 → 내보내기 만들기 → 기기로 내보내기 → 정보 맞춤 설정
    → '팔로워 및 팔로잉'만 선택
  3. 기간: 전체 기간 (필수!)
  4. 형식: JSON (필수!)
  5. 5분 정도 기다렸다가 파일 다운로드
    (※ 팔로워/팔로잉 수가 많을 경우 시간이 더 걸릴 수 있습니다.)
    🚫 압축은 절대 풀 필요가 없습니다! (ZIP 파일 그대로 사용)
    🌐 파일 업로드 오류 방지를 위해 꼭 기본 인터넷(사파리, 크롬)에서 진행해 주세요.
💡 앱에 뜨는 숫자랑 결과가 다른가요?

앱에서는 숨겨지는 '비활성화/정지/탈퇴 계정'들이 원본 데이터(JSON)에는 그대로 남아있기 때문입니다. 오류가 아니니 안심하세요!

💡 다운받은 파일을 양쪽 모두에 올려주세요!
어느 쪽이든 같은 파일(ZIP 또는 JSON)을 넣으면 알아서 분석합니다.
